# Learners School : 통계 기초

통계를 처음 배우는 학습자가 데이터 요약, 확률, 표집, 추정, 가설검정, 회귀, 실험 설계까지 단계적으로 익힐 수 있도록 구성한 실무형 입문서 프로젝트입니다.  
공식을 외우는 방식보다, 데이터에서 불확실성을 어떻게 읽고 의사결정으로 연결하는지에 초점을 둡니다.

## 이 책이 다루는 것

- 모집단, 표본, 변수, 척도 같은 통계의 기본 언어
- 평균, 분산, 표준편차, 사분위수, 왜도, 이상치 같은 기술통계
- 사건, 조건부확률, 베이즈 정리, 확률분포 같은 확률 기초
- 표본분포, 중심극한정리, 신뢰구간, 점추정과 구간추정
- 귀무가설, p-value, 1종 오류, 2종 오류, 검정력, 주요 검정 방법
- 상관분석, 회귀분석, A/B 테스트, ANOVA
- 통계 결과를 잘못 해석하는 대표적인 실무 사례와 보고 방법

## 이런 분께 적합합니다

- 통계를 처음 배우는 분
- 데이터 분석, BI, 대시보드 업무를 시작하는 분
- SQL, Tableau, Python은 다루지만 통계 해석이 불안한 분
- A/B 테스트, 실험, 모델 성능 보고서를 더 정확하게 읽고 싶은 분

## 학습 목표

이 책을 마치면 다음 내용을 스스로 설명하고 적용할 수 있습니다.

- 데이터 요약 지표를 상황에 맞게 선택하고 해석하기
- 표본과 모집단의 차이를 이해하고 추정 결과를 과장하지 않기
- p-value, 신뢰구간, 검정력을 혼동하지 않고 읽기
- 상관관계와 인과관계를 구분하며 결과를 해석하기
- 실험 결과를 통계적으로 검토하고 실무 의사결정으로 연결하기

## 도서 구성

### 1. 통계적 사고의 출발점

통계가 왜 필요한지, 그리고 모집단, 표본, 변수, 척도 같은 기본 언어를 정리합니다.  
이 장을 통해 이후 수식이 단순 기호가 아니라 데이터 구조를 설명하는 도구라는 점을 이해합니다.

### 2. 데이터 요약과 분포

평균, 중앙값, 표준편차, 사분위수, 왜도, 이상치를 중심으로 데이터를 요약하는 방법을 배웁니다.  
같은 평균이라도 분포가 전혀 다를 수 있다는 점을 실무 예시와 함께 설명합니다.

### 3. 확률 기초

사건, 확률, 조건부확률, 베이즈 정리, 확률변수와 분포를 다룹니다.  
가설검정과 머신러닝 평가 지표를 이해하기 위한 토대를 만드는 장입니다.

### 4. 표집과 추정

표본이 어떻게 모집단 정보를 담는지, 그리고 그 과정에서 어떤 오차와 편향이 생기는지 설명합니다.  
표본분포, 중심극한정리, 신뢰구간 해석까지 연결합니다.

### 5. 가설검정

귀무가설과 대립가설, p-value, 유의수준, 검정력의 관계를 정리합니다.  
왜 “유의하다”가 곧 “중요하다”를 뜻하지 않는지도 함께 다룹니다.

### 6. 상관과 회귀

변수 사이의 관계를 정량화하는 상관분석과 회귀분석을 다룹니다.  
계수 해석뿐 아니라 선형성, 잔차, 다중공선성 같은 진단 관점도 소개합니다.

### 7. 실험 설계와 분산분석

A/B 테스트와 ANOVA를 중심으로 여러 그룹 비교와 실험 설계의 핵심을 정리합니다.  
실무에서 실험 결과를 잘못 읽게 되는 이유도 함께 짚습니다.

### 8. 실무에서 자주 하는 통계 해석 실수

상관관계를 인과관계로 착각하는 문제, 표본 편향을 무시하는 문제, 보고서에서 숫자를 과장하는 문제를 정리합니다.  
좋은 분석 결과를 만드는 것만큼, 좋은 해석과 커뮤니케이션이 중요하다는 점을 강조합니다.
